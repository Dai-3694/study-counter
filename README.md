# べんきょうタイムテレビ

子どもが勉強した時間を、その日のテレビ時間として使えるようにする家族向けPWAです。Androidタブレットやスマートフォンのホーム画面に追加して、シンプルなタイマーアプリとして使えます。

## できること

- 勉強時間を秒単位でカウントできます。
- 調べ物などで別アプリや別タブを開いても、復帰時に経過時間を反映します。
- 勉強タイマーは一時停止・再開できます。
- 勉強終了後に、子どもがパパ・ママへ成果を見せる確認ステップがあります。
- パパ・ママは成果に合わせて、テレビ時間を5分単位で増減できます。
- 決定したテレビ時間をカウントダウンできます。
- 残っているテレビ時間に、次の勉強で獲得した時間を加算できます。
- テレビタイマーは一時停止・再開・リセットできます。
- テレビ時間が残っている状態で勉強を始めると、加算するかリセットするか選べます。
- テレビ時間が0になったら音で知らせます。
- PWAとしてホーム画面追加とオフライン利用に対応しています。

## 使い方

1. 勉強を始めるときに「勉強スタート」を押します。
2. 途中で休むときは「一時停止」を押します。再開するときは「勉強を再開」を押します。
3. 勉強が終わったら「勉強おわり！」を押します。
4. ノートやプリントなど、がんばった成果をパパ・ママに見せます。
5. パパ・ママが内容に合わせてテレビ時間を調整し、「この時間で決定」を押します。
6. テレビを見るときに「テレビを見る」を押すと、残り時間のカウントダウンが始まります。
7. 翌日に持ち越さないときなどは「今日のテレビ時間をリセット」でテレビ時間をゼロにできます。
8. 残り時間が0になると、テレビ時間終了の音が鳴ります。

## バックグラウンド中の動き

このアプリのタイマーは、画面更新の回数ではなく「開始した時刻」と「現在の時刻」の差分で計算します。そのため、勉強中にタブレットで調べ物をしたり、別タブへ移動したりしても、アプリに戻ったときに経過時間が反映されます。

画面を離れる直前や復帰時にも状態を保存・再計算するため、ブラウザがバックグラウンド中に画面更新を止めた場合でも、勉強時間やテレビ残り時間が大きくずれにくい作りです。

## テレビ時間の扱い

- テレビ時間が残っている状態で新しく勉強を始めると、残り時間に加算するか、リセットして始めるかを選べます。
- 加算を選んだ場合、新しく獲得したテレビ時間は、残っているテレビ時間に加算されます。
- リセットを選んだ場合、残っているテレビ時間をゼロにしてから勉強タイマーを開始します。
- テレビタイマー中に勉強を始めることはできません。一時停止中または停止中の残り時間は、勉強開始時に扱いを選べます。
- 「今日のテレビ時間をリセット」を押すと、確認後にテレビ時間をゼロにできます。

## ローカルで起動する方法

Node.js が使える環境で、以下を実行してください。

```bash
npm start
```

起動後、ブラウザで次のURLを開きます。

```text
http://localhost:8000
```

## テスト方法

JavaScriptの構文チェックを実行します。

```bash
npm test
```

## ビルド方法

配信用ファイルを `dist/` にコピーします。

```bash
npm run build
```

## GitHub Pagesで公開する方法

GitHubリポジトリの `Settings > Pages` で、以下のように設定してください。

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

`main` ブランチにマージされると、静的ファイルとしてそのままGitHub Pagesで公開できます。

## 主なファイル

- `index.html`: 画面構成
- `styles.css`: デザイン
- `app.js`: タイマーや保存処理などのアプリ本体
- `sw.js`: オフライン利用のためのService Worker
- `manifest.webmanifest`: PWA設定
- `assets/`: アプリアイコン
